Store Integer Values from Sensor to DataBase


Discription: 
it`s a web page with GET method that gets values from sensor and store them in DataBase.


Languages and Technologies: 
Node.js for back-end. 
Express framwork for web application
MySQL for database.
Arduino to send request from ESP32.


Environemt: 
ESP32 that connected to Internt with WIFI.	


How it will work:
Firsly, ESP32 will be connected with Internet through wifi. then send integer value via http request with GET method. lastly, this value will be stored in DataBase that will be displayed as a response for the request.
