Control The Movements of Robot and Store them in DataBase


Discription: 
it`s a web page that has control panel to control robot movements. 

Languages and Technologies: 
Node.js for back-end. 
Express framework for web application
MySQL for database.
Arduino to get the movements request.


Environemt: 
you need to install express package by using this command: 
	npm install express	
make sure to run the server(backend code) befor you run the webpage


How it will work:
after you run the server, you can press on any button direcion you want to make robot move. you will alert you with the type of movement.
